# revcd.com

Codebase for my personal website.

## Development

Install [Hugo](https://gohugo.io/installation), then run:

```bash
# Install deps.
yarn

# Build Tailwind styles.
yarn tw:watch

# Run local Hugo server.
make run # → http://localhost:5173
```
